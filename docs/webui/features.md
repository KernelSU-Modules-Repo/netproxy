# WebUI 功能详情

## 状态页

仪表盘页面，支持自定义布局。

### 仪表盘卡片

| 卡片 | 说明 |
|------|------|
| 网络速度 | 实时上传/下载速度图表 |
| 出站模式 | 规则/全局/直连 模式切换 |
| 网络检测 | 出口 IP 地址（点击刷新） |
| 内网 IP | 本机内网 IP 地址 |
| 流量统计 | 上传/下载流量环形图 |
| CPU | 系统 CPU 使用率 |
| 内存 | 系统内存使用率 |

### 仪表盘自定义

- 点击右上角编辑按钮进入编辑模式
- 拖拽卡片调整顺序
- 点击 × 隐藏卡片
- 点击"添加组件"显示隐藏的卡片
- 布局自动保存到本地

---

## 节点页

管理代理节点和订阅。

### 节点操作

- **添加节点**：导入节点链接（vless/vmess/trojan/ss/hysteria2）
- **添加出站**：导入完整的 Xray outbound JSON
- **添加订阅**：输入订阅名称和 URL

### 订阅管理

- 订阅节点按分组显示（标签页）
- 点击订阅名右侧菜单可更新/删除订阅
- 支持 Clash 订阅格式自动转换

### 节点切换

- 点击节点卡片切换当前节点
- 当前节点显示高亮状态
- 服务运行时支持热切换

---

## 应用页

分应用代理设置。

### 代理模式

| 模式 | 说明 |
|------|------|
| 关闭 | 不启用分应用代理，所有应用都走代理 |
| 白名单 | 仅选中的应用走代理 |
| 黑名单 | 选中的应用不走代理 |

### 应用列表

- 显示已安装应用（带图标）
- 支持搜索过滤
- 勾选应用添加到列表
- 已选应用在列表顶部显示

---


## 设置页

模块配置管理。

### 基本设置

- 开机自启开关
- 服务启动/停止/重启按钮

### 高级设置

- TProxy 配置
- 延迟测试目标设置

### 路由规则

- 规则列表（启用/禁用）
- 规则排序（拖拽）
- 导入 Clash 规则集

### 外观设置

- 主题模式（自动/浅色/深色）
- 语言设置（中文/English）

### 关于

- 模块版本信息
- 更新检查
- GitHub 链接
