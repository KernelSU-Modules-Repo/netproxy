# 快速开始

本指南帮助您在 5 分钟内完成 NetProxy 的基本配置。

## 第一步：添加节点

有两种方式添加代理节点：

### 方式一：导入节点链接

1. 打开 WebUI，进入 **节点** 页面
2. 点击右上角 **导入链接** 按钮
3. 粘贴节点链接（支持 vless://、vmess://、trojan:// 等）
4. 点击 **导入**

### 方式二：添加订阅

1. 进入 **节点** 页面
2. 点击右上角 **添加订阅** 按钮
3. 输入订阅名称和 URL
4. 点击 **添加**，等待导入完成

## 第二步：选择节点

1. 在节点列表中找到要使用的节点
2. 点击节点卡片进行切换
3. 当前选中的节点会显示高亮状态

## 第三步：启动代理

1. 进入 **状态** 页面（首页）
2. 点击大的 **启动/停止** 按钮
3. 服务启动完成
