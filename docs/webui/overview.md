# WebUI 概览

NetProxy 内置 Web 管理界面，基于 MDUI 2 (Material Design 3) 构建。

## 访问方式

1. 打开 KernelSU/Magisk 应用
2. 进入 **模块** 页面
3. 找到 NetProxy 模块
4. 点击右侧的 **WebUI** 图标

## 界面结构

WebUI 包含五个主要页面，通过底部导航栏切换：

| 页面 | 说明 |
|------|------|
| 状态 | 仪表盘，显示运行状态和网络信息 |
| 节点 | 节点管理、订阅管理 |
| 应用 | 分应用代理设置 |
| 设置 | 模块配置、路由规则 |

---

## 技术栈

- **UI 框架**：MDUI 2 (Material Design 3)
- **开发语言**：TypeScript
- **图表库**：uPlot（网络速度图表）
- **构建工具**：Vite
- **主题**：支持 Monet 动态取色

## 主题支持

- **自动模式**：跟随系统 + Monet 动态取色
- **浅色模式**：固定浅色主题
- **深色模式**：固定深色主题
